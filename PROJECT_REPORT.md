# Mini Project Report

## Title
Traffic Sign Recognition Using Group Convolutional Neural Networks

## Problem statement
Traffic sign recognition systems used in intelligent transportation must classify signs accurately while responding quickly. Conventional CNNs can be accurate but computationally expensive, while highly compressed approaches such as binary neural networks may sacrifice accuracy. The project studies grouped convolution as an efficiency-oriented alternative.

## Proposed solution
The proposed GCNN uses pointwise convolutions for channel projection and mixing, grouped 3×3 convolutions for lower computational cost, batch normalization, ReLU activations, residual connections and global average pooling. It predicts one of 43 GTSRB classes.

## Modules
1. Dataset acquisition and preprocessing
2. Data augmentation
3. GCNN model construction
4. Baseline CNN construction
5. Training and validation
6. Test evaluation
7. Single-image inference
8. Complexity and latency benchmarking

## Evaluation metrics
- Classification accuracy
- Precision, recall and F1-score
- Trainable parameter count
- Multiply-accumulate operations
- Average inference latency per image

## Expected conclusion
The experiment should determine whether grouped convolution reduces computational cost and latency while preserving competitive recognition accuracy. Final claims must be based on measurements generated by the included scripts.
